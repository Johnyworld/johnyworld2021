# Johnyworld 2021
The new version of johnyworld.com that developer Johny Kim's portfolio web site.

## Dependencies
- React (Hooks) (CRA)
- Redux Toolkit
- Typescript
- Styled Components
- i18next
- HTML5 Canvas
- Node JS
- Express
- Socket.io
- MariaDB : AWS RDS
- AWS S3 Storage


## Install

#### server

- Node v12.18.0 or higher must be installed.
- Nodemon v2.0.4 or higher might be installed.

```
cd server
yarn
or
npm install
```

#### client

```
cd client
yarn
or 
npm install
```

## Start Development

You should start dev server first to use proxy setting.

#### server

```
cd server
yarn dev
or 
npm run dev
```

#### client

```
cd client
yarn start
or
npm run start
```

#### by Docker

```
docker-compose up -d --build
```
