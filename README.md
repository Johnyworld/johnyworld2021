### Johnyworld 2021
웹개발자 Johny Kim의 홈페이지를 리뉴얼합니다.

### Dependencies
- React (Hooks) (CRA)
- Typescript
- Styled Components
- i18next
- HTML5 Canvas
- Node JS
- Express
- Socket.io
- MariaDB : AWS RDS
- AWS S3 Storage


### Install
```
yarn global add nodemon
or
npm install -g nodemon

/server
yarn
or
npm install

/client
yarn
or 
npm install
```

### Start Development
```
/server
yarn dev
or 
npm run dev

/client
yarn start
or
npm run start
```
