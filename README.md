### Johnyworld 2021
The new version of johnyworld.com that developer Johny Kim's portfolio web site.

### Dependencies
- React (Hooks) (CRA)
- Redux Toolkit
- Typescript
- Styled Components
- i18next
- HTML5 Canvas
- Node JS
- Express
- Socket.io
- MariaDB : AWS RDS
- AWS S3 Storage


### Install
```
yarn global add nodemon
or
npm install -g nodemon

/server
yarn
or
npm install

/client
yarn
or 
npm install
```

### Start Development

You should start dev server first to use proxy setting.

```
/server
yarn dev
or 
npm run dev

/client
yarn start
or
npm run start
```
